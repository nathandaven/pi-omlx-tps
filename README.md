# pi-omlx-stats

This extension is basic and adds omlx stat reporting using the built in omlx stats endpoint. The goal was to bring the stats given by the webui to the processing prompt.

Example:

```sh
 ⠼ Qwen3.6-35B-A3B-MLX-oQ4-FP16 · 1 PP · 1 req · ↓27 tok/s | 2% (521/32768) 19.5s | 1208.9s | ~19.20GB / 19.92GB MEM 19.7GB / 21.9GB
```

It is reccomended t ouse the other omlx extension for the model configuration.

### Install

```sh
pi install npm:pi-omlx-stats
```
